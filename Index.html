¿Qué es Git?

Git es un sistema de control de versiones distribuido, usado principalmente para gestionar el desarrollo de software. Te permite registrar los cambios de un proyecto a lo largo del tiempo, facilitando la colaboración entre múltiples desarrolladores.

Instalación de Git

Para instalar Git, puedes seguir los pasos según tu sistema operativo:

  • Windows: Descarga el instalador desde el sitio oficial de Git.
  • macOS: Usa Homebrew: brew install git
  • Linux: Usa el gestor de paquetes de tu distribución, por ejemplo: sudo apt install git en Ubuntu.

Comandos Básicos de Git

  • git init: Inicializa un nuevo repositorio Git en un directorio.
  • git clone [URL]: Clona un repositorio remoto.
  • git status: Muestra el estado actual del repositorio (archivos modificados, sin seguimiento, etc.).
  • git add [archivo]: Añade un archivo al área de preparación.
  • git commit -m "[mensaje]": Crea un nuevo commit con los cambios preparados.
  • git push: Sube los cambios locales al repositorio remoto.
  • git pull: Actualiza el repositorio local con los cambios del remoto.
  • git branch: Lista, crea o elimina ramas.
  • git checkout [rama]: Cambia a una rama específica.
  • git merge [rama]: Fusiona una rama en la rama actual.
