# Pràctica Grupal GIT

## Descripció

Aquesta és una pràctica grupal creada per a comprendre millor el funcionament de Git i les seves branques. A través d'aquesta activitat, explorarem com les branques interactuen entre elles quan fem canvis en els diferents arxius del projecte.

### Contingut del projecte
- **Index.html**: L'arxiu principal HTML on estructurem la nostra pàgina web.
- **index.css**: L'arxiu CSS per a donar estil a la nostra pàgina.
- **scripts/Agax.js**: Un script de JavaScript que controla les funcionalitats dinàmiques.
- **README.md**: Aquest document d'explicació del projecte.

## Instruccions

1. **Clona** aquest repositori al teu ordinador.
2. **Crea** una nova branca per a fer els teus canvis.
3. **Modifica** els arxius HTML, CSS o JavaScript per a afegir o canviar funcionalitats.
4. **Commita** i **puja** els teus canvis a la teva branca.
5. **Fes una fusió** entre les diferents branques per veure com es resolen els conflictes.

## Contribució

Per a contribuir al projecte, assegura't de seguir les bones pràctiques amb Git:
- Treballa sempre en branques separades.
- Comenta clarament els teus canvis en cada commit.
- Si detectes un conflicte durant la fusió, intenta resoldre'l abans de fer el merge final.
